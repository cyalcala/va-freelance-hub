# VA.INDEX Sovereign Ethics Protocol
## Version: 1.0
## Date: 2026-04-11

This protocol governs the "Space Age" acquisition strategy of VA.INDEX. It defines how we maintain the technical and moral higher ground while achieving operational data dominance.

---

## 1. The Mirror Principle (Transparency)
We move from "Scraping" to "Mirroring." 
- **Method**: We treat Search Engine Results Pages (SERPs) as the definitive public mirror of the internet. 
- **Rule**: If a direct target (e.g. LinkedIn, X) blocks scrapers, we do not bypass their technical barriers directly. We instead query the public search indices (Google/Bing) which have already been authorized to index that content. This ensures we are always "reading from the mirror" rather than "invading the room."

## 2. The Efficiency Mandate (Empathy for Servers)
We treat the target's infrastructure with respect.
- **Method**: Prioritize **Headless APIs** (Vector 5) over UI Rendering.
- **Rule**: A single API fetch cost vs. a full Playwright/Puppeteer UI render is a 100x reduction in server load. We are "Good Citizens" who minimize the technical tax we place on job boards and agency servers.

## 3. The PII Quarantine (Privacy)
We are an Opportunity Index, not a People Index.
- **Rule**: We never store Personal Identifiable Information (PII) of non-recruiting employees. We only harvest **Hiring Signals** (Company, Title, Role, Opportunity URL).
- **Rule**: If a scraper accidentally captures a resume or a personal profile, it must be purged at the sifting layer before it reaches the Gold Vault.

## 4. The $0 Community Mandate (Sustainability)
Sustainability is a moral choice.
- **Rule**: To ensure the platform remains a public utility, it must cost $0 to operate. We reject expensive "Pro" proxies and paid APIs. We achieve scale through **Intelligence** (Distributed GitHub Actions, Semantic Dorking) rather than **Power** (Paid Proxies).

## 5. Non-Intrusive Discovery
- **Rule**: Never bypass authentication (Login Walls). If data requires a login, it is not "Publicly Available" for the purposes of this index. We only index the "shared web."
